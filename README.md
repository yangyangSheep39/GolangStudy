### 拉不下来包

```Go
export GOPATH=/usr/local/go/bin
export GOPROXY=https://goproxy.io
export PATH=$PATH:$GOPATH
export GO111MODULE=on
```

### 开启module

```Go
go env -w GO111MODULE=on
```

### 设置代理

```Go
go env -w GOPROXY=https://goproxy.io,direct
```
